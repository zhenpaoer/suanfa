二维与一维对照关系

dp[i][j] = dp[i][j-1] + dp[i-1][j] + dp[i-1][j-1]
            左边           上边           斜对角
dp[j]    = dp[j-1]    + dp[j]      + temp 没有斜对角，只能将其存在临时变量里
temp其实就是上个行循环的dp[i]
